What is Sketch?

Sketch is a modern design application for modern graphic designers.

Sketch feels just like that. It feels like an application built for the way that we're building sites and applications today. Our sites need to be flexible, responsive, and work across multiple screen sizes and devices. Sketch helps us work in this way more efficiently.

What is Sketch Good For?

You can use Sketch for the same things you'd use any design application.

Things like:

  • User Interface Design
  • Web and Application Design
  • Mobile Application Design
  • Illustration

I really like this quote from Sketch's site. This plays back into what we were talking about before with Sketch being a modern design application. It just works how we work.

And a lot of it has to do with the Sketch community. There are a lot of designers that are contributing to the Sketch ecosystem, and it helps to improve the application.

This is what the Sketch interface looks like. It's comprised of four parts.

The big white are is the Canvas, and it's where you will work. The Canvas is infinite, but you can defined confined spaces, and we'll talk about that in a bit.

The top bar is the Toolbar, and it has a lot of useful options for creating and editing shapes. If you want to customize what's in this bar, right-click on the Toolbar and select Customize Toolbar.

Next we have the Layer List, which functions just how it would in Photoshop or Illustrator. All of your layers will show up here, and you can group, sort, and search any of your layers.

Finally, we have the Inspector, which is where you will spend most of your time. In this panel, you will edit your text and shapes, adding colors, borders, blending modes, and a whole lot more.

Primary Features

So now, let's talk about some of the primary features of Sketch.

Vector

First off, Sketch is entirely vector. Everything you create will scale to any size, just like any vector would.

We can see here that we've drawn a shape, and while we're zoomed in to 800%, the shape is still nice and crisp.

However, in Sketch, you can also opt to see this vector shape in a Pixel View, which will make the shape appear bitmapped. This is useful if you want to see the pixels, and make very detailed adjustments to your design.

Symbols

One of my favorite Sketch features is Symbols, which are new to Sketch 3. You can think about Symbols in Sketch like you would vector smart objects in Photoshop. Essentially, you can create an object, duplicate it, and whenever you alter any instance of that object, they all change.

Text Styles

Similar to the concept of Symbols are Text Styles. With Text Styles, you can set specific styles on your text elements, and easily add that style across your document. I like to think of it like headings in HTML. You have an <h1> with certain styles, an <h2> with certain styles, and so on and so forth.

Demo Time!

A short demo of Text Styles in Sketch.

Artboards

Artboards in Sketch are just like those in Illustrator. You can create an artboard at a set size within your Canvas to work. Additionally, in Sketch, when you select to create an artboard, you have the ability to chose from a set of preset templates, such as an iOS icon, a desktop browser size, iPhone/iPad size, as well as other standard presets.

PAGES

Pages are somewhat similar to Layer Comps in Photoshop, but much much better. With Pages, you can easily create multiple documents within your document, and easily use things like Symbols across all pages within a document.

Auto Save & Versions

Auto Save and Versions are more primary features that are super useful in Sketch.

Auto Save

As you'd expect, Auto Save will automatically save out versions of your document as you work.

When you go to File > Revert To, you can see the saved versions, and revert, if need be.

Export

Keeping with the theme of Sketch being a modern design application, the Export feature really plays to that.

Here we have a few different shapes we've created.

And you can see this Make Exportable option down in the bottom right of the Inspector.

When we click it, we get the option to export a 1x PNG image. I did say that Sketch is a modern design application, right? So let's do more.

When we click the plus at the top right, we now get a second export, which is a 2x (two times the size) PNG. So, we're saving out a regular 1x version, and a retina-ready version all in the same pass.

If we click the plus again, we can even save out a different filetype (in this case, an SVG).

Now we have our two PNGs, as well as an SVG, all exported in one pass.

Sketch Mirror

Sketch Mirror is an iOS application that you can connect to an instance of Sketch running on your Mac, and then live design and preview right on the device.

You can to create multiple Artboards and swipe between them on your device.

And you can even preview both iPhone and iPad Artboards on the respective device at the same time.
